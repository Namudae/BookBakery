# 책빵(Spring-Project)
>  ## :page_with_curl:개요 및 기획의도
 - Spring FrameWork를 사용한 팀 프로젝트입니다.

### Why:question:
- 여러 도서판매 사이트를 보면서 도서판매 뿐 아니라 사이트를 이용하는 사용자들을 위한 커뮤니티 게시판과 서로 중고 도서를 거래하는 기능이 있으면 좋을 것 같다고 생각했습니다.
- 
### So:exclamation:
- 헌책방이라는 게시판을 만들어 사용자들끼리 중고거래 하는 기능과 채팅 기능을 포함해 거래의 편의성을 높였습니다.
- 북유럽이라는 커뮤니티 게시판을 통해 책의 서평을 등록하고 사용자들끼리 소통할 수 있는 공간을 만들었습니다.
